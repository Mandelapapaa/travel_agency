# Safari Travel Management System

A web-based **Safari Travel Management System** developed using **Yii2 Framework** and **MySQL**.

The system helps manage travel agency operations including customers, tour packages, bookings, payments, and administrator accounts.

---

## Features

### Customer Management
- Add and manage customer information
- View customer records
- Update and delete customer details

### Tour Package Management
- Create safari packages
- Manage destinations
- Manage package duration and prices

### Booking Management
- Create customer bookings
- Assign tour packages to customers
- Track booking dates
- Manage booking status

### Payment Management
- Record customer payments
- Track payment methods
- Monitor payment transactions

### Administrator Management
- Manage system administrators
- Secure administrator accounts
- Control system access

## System Architecture

The project follows the **Model View Controller (MVC)** architecture provided by Yii2.

### Model
Handles:
- Database interaction
- Data validation
- Business logic

### View
Handles:
- User interface
- Displaying information
- Forms and tables

### Controller
Handles:
- User requests
- Communication between Model and View

---

## Database Tables

The system contains the following main tables:

- Users
- Customers
- Packages
- Bookings
- Payments
- Admins
